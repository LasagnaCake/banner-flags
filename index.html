How to add the banner to your page

For tumblr

If done via this method, you MUST change the height="100%" at the top of the file to your desired height!
  1. Open SVG file on text editor;
  2. Copy the entirety of its contents;
  3. Open blog customization;
  4. On your theme, click "Edit HTML";
  5. Make a backup of your theme in case something goes wrong;
  6. Paste the SVG's contents somewhere within the <body></body> tag (preferably before any other page content);
  7. Done!

For custom website

If you don't feel like adding a file, you can do the same way as for tumblr - via copying the file's contents directly into the page.
  1. Add SVG file somewhere in website directory;
  2. Add <object data="path/to/file.svg" type="image/svg+xml" width="100%" height="{YOUR_PREFERRED_HEIGHT}"></object> within the <body></body> tag (preferably before any other page content);
  3. Done!
